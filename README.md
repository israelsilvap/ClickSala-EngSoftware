## Como utilizar

```bash
    # Clone o projeto
    $ git clone https://github.com/IsraelS1lv4/ClickSala-EngSoftware.git
```

```bash
    # Entre no diretório
    $ cd frontend_clickSalas
```

```bash
    # Instale as dependências
    $ npm install
```
